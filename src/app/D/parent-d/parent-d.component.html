Event Bubbling

Event bubbling is a concept in JavaScript (and consequently in Angular, which is built on JavaScript) where an event that is triggered on a nested element is propagated up through its ancestor elements in the DOM hierarchy. This allows you to handle the event at multiple levels in the DOM tree.

Capturing Phase: In this phase, the event starts from the root of the DOM tree and travels down to the target element. Event handlers attached at the higher level of the DOM tree are invoked first, and as the event descends, it reaches the target element.

Capturing/Trickling

Event delegation is a technique in web development where a single event handler is attached to a common ancestor of multiple elements, rather than attaching individual event handlers to each element. This is especially useful when you have a large number of similar elements (e.g., a list of items) and you want to handle events for all of them efficiently. In Angular, event delegation can be implemented using event binding and leveraging the event object to determine which specific element within the common ancestor was interacted with.
